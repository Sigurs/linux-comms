## ADDED Requirements

### Requirement: Open profile menu via right-click
The system SHALL open the profile options context menu when the user right-clicks a profile entry in the sidebar.

#### Scenario: Right-click opens context menu
- **WHEN** the user right-clicks a profile icon in the sidebar
- **THEN** the profile context menu SHALL appear at the cursor position
- **AND** the default browser/Electron context menu SHALL be suppressed
